About Me

Jmenuji se Filip a jsem cílevědomý začínající vývojář, který se zaměřuje na backend a full-stack programování. Aktivně se učím technologie jako JavaScript, React, Python, MySQL, .NET Core, C#, Spring Boot a ASP.NET a tvořím reálné projekty, které rozšiřují mé znalosti a portfolio. Zajímám se nejen o software, ale i o techniku, vývoj chytrých zařízení a propojení umělé inteligence s hardwarem. Mým dlouhodobým cílem je vytvářet inovativní technologie, které mají pozitivní dopad na svět. Do každého projektu přináším disciplínu, touhu růst a řešit skutečné problémy. Ať už jde o backendovou logiku, API nebo webové aplikace, vždy se snažím psát čistý a efektivní kód s reálným účelem.
